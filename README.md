## Installation

### Without publishing to the VS Code Marketplace

```
ln -s /path/to/ronny.birds-of-paradise/ $HOME/.vscode/extensions/ronny.birds-of-paradise/
```

Trigger `Developer: Reload Window`.

Then change the color theme.
